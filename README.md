# CampusCare

CampusCare is a college-based complaint and issue management website designed to help students easily report problems happening inside the campus such as cleanliness issues, water problems, electricity faults, classroom maintenance, hostel issues, and security-related complaints.

This website provides a simple and user-friendly platform where students can register complaints online, upload complaint details, and track the complaint status using a complaint ID. It reduces the need for students to visit offices physically and helps the administration manage complaints faster.

The admin panel allows college authorities to view all complaints, update their status (Pending / In Progress / Resolved), and respond efficiently. CampusCare improves transparency, saves time, and ensures faster resolution of campus-related issues.

Key Features
Student complaint submission form
Complaint tracking system
Admin dashboard for complaint management
Status update system
Secure database storage (Firebase / Backend)
